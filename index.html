Earthquakes (map dashboard)

This one-page dashboard demonstrates some of my JavaScript skills for data extraction and visualisation:

  • I use D3.js to retrieve earthquakes data from the USGS API in GeoJSON.
  • The map and markers are created with Leaflet.js

The dashboard is simple, easy to use and responsive. While the data management is very straightfoward thanks to a reliable and well-organised source, most of the effort went into the visualisation itself. I spent time on the best way to display the important information:

  • markers size for the magnitude
  • colour for the depth
  • overlay for more details about the location and exact magnitude and depth values
  • use of an additional layer to display the tectonic plates boundaries
  • unique colour palette and bespoke CSS for the overall style of the website
